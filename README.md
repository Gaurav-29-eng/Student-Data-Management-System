# Student Data Management System

## Description

A simple console-based Python application to manage student records.

## Features

* Add student
* View all students
* Search student by roll number
* Update student details
* Delete student

## Technologies Used

* Python
* Data Structures (List, Dictionary)

## How to Run

1. Open terminal
2. Run: python filename.py

## Learning Outcome

* Improved understanding of Python basics
* Learned data handling using lists and dictionaries
* Practiced modular programming using functions
